# CGPA Calculator 🎓

A console-based Java program that calculates CGPA 
based on grade points and credit hours.

## How It Works
- Enter number of subjects
- Enter grade points (0.0–4.0) and credit hours per subject
- Program calculates your weighted CGPA

## Grade Scale (Pakistani Universities)
| Grade | Points |
|-------|--------|
| A     | 4.0    |
| A-    | 3.7    |
| B+    | 3.3    |
| B     | 3.0    |
| B-    | 2.7    |
| C     | 2.0    |
| F     | 0.0    |

## How to Run
1. Clone this repo
2. Open in IntelliJ IDEA
3. Run `CGPACalculator.java`

## Built With
- Java 21
- IntelliJ IDEA
